## Descripción
el sistema esta pensado para la gestion de la produccion en planta de una empresa.
El sistema en su forma más básica. Consta de tres secciones: oficina técnica, supervisores y operario y un home con la autentificación.
Estas son las funciones mínimas para el sistema, pero la idea es que sea modularizable y poder adaptarse a cualquier empresa según lo que se desee controlar. De esta forma todos los procesos de un producto en la planta quedan registrados y almacenado el quien, el que, y el cómo.

## Tecnologías
- API Restful en node.js y Express
- Front en React
- Base de datos en MySQL
- Control de versiones: Git/ Github
